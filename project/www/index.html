Добро пожаловать на демонстрационную страницу Ethics Skeleton

Что необходимо:

  • Node.js + NPM
  • Git-client

Этапы установки:

  1. Склонировать проект с GitHub git clone git://github.com/TimurSeyidov/ethics-skeleton.git
  2. Запустить комманднкую строку из директории с проектом
  3. Выполнить комманду npm install
  4. После установки пакетов запускаем комманду gulp (должен запуститься сервер на порту 7700)
  5. Также доступна комманда gulp compress, которая сожмет Ваши css и js файлы
